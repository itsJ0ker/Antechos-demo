## Data Flow

```
User Action вҶ’ State Update вҶ’ Re-render вҶ’ Display Update

Examples:
1. Click Next Button
   вҶ’ currentIndex += 4
   вҶ’ Display next 4 universities

2. Click View All
   вҶ’ showAll = true
   вҶ’ Display all 19 universities in grid

3. Click University Card
   вҶ’ window.open(university.link)
   вҶ’ Opens application page in new tab

4. Click Pagination Dot
   вҶ’ currentIndex = dotIndex * 4
   вҶ’ Jump to specific page
```
