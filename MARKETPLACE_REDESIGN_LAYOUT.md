## Color Scheme

### Default Colors
- **Primary**: Blue (#3b82f6)
- **Secondary**: Indigo (#6366f1)
- **Accent**: Purple (#8b5cf6)
- **Success**: Green (#10b981)
- **Warning**: Orange (#f59e0b)
- **Danger**: Red (#ef4444)

### Backgrounds
- **White**: #ffffff
- **Gray 50**: #f9fafb
- **Gray 100**: #f3f4f6
- **Blue 50**: #eff6ff

### Text
- **Primary**: Gray 900 (#111827)
- **Secondary**: Gray 600 (#4b5563)
- **Muted**: Gray 500 (#6b7280)

## Typography

### Headings
- **H1**: 3xl-5xl (Hero titles)
- **H2**: 2xl-3xl (Section titles)
- **H3**: xl-2xl (Subsection titles)
- **H4**: lg-xl (Card titles)

### Body Text
- **Large**: text-lg (Hero descriptions)
- **Normal**: text-base (Body content)
- **Small**: text-sm (Captions, labels)
- **Tiny**: text-xs (Badges, tags)

## Spacing

### Sections
- **Padding Y**: py-16 (64px)
- **Padding X**: px-6 (24px)
- **Max Width**: max-w-7xl (1280px)

### Cards
- **Padding**: p-6 to p-8
- **Gap**: gap-6 to gap-8
- **Rounded**: rounded-2xl

### Grids
- **Gap**: gap-6 to gap-8
- **Columns**: 1 → 2 → 4

```
rounded-2xl shadow-lg

```

## Animations

### Framer Motion
- **Initial**: opacity: 0, y: 20
- **Animate**: opacity: 1, y: 0
- **Transition**: delay based on index

### CSS Animations
- **Testimonials**: Scroll up (20s linear infinite)
- **Hover**: Scale, shadow, border color
- **Donut Charts**: Stroke animation (1s)
